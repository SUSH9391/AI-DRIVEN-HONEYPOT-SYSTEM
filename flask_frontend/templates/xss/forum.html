Always validate on the server side. Client-side validation is just for UX (giving fast feedback to the user). An attacker can easily bypass client-side checks.

Here is a quick example of what NOT to do:

app.post('/api', (req, res) => {
  // Never trust user input directly!
  db.query(`SELECT * FROM users WHERE id = '${req.body.id}'`);
});
